# 2020 12月 3：布拉索夫中央广场的圣诞市集，罗马尼亚

# 喀尔巴阡式的圣诞庆典（© Alpineguide/Alam）

坐落在坦帕山在罗马尼亚喀尔巴阡山脉南部,胸罩șov中世纪的欧洲是一个很好的例子的交叉文化。由日耳曼骑士建立，撒克逊人和匈牙利人定居，受奥斯曼帝国、鞑靼人和罗马尼亚本地人的影响，这座城市的哥特式建筑和独特的饮食反映了它丰富的历史。该市的圣诞市场在整个12月都在举行，主要是一棵27米高的活圣诞树。市场上众多的摊位以食物、饮料和当地工艺品为特色。这个节日被称为“胸罩şov -这座城市从故事的,也许是因为传说这广场哈梅林的孩子被带走后出现魔笛。无论你到哪里，这都是一个神奇的地方，你可以在这里喝一杯祝酒，用热情的“Noroc!”向罗马尼亚主人敬酒。

# 2020 12月 5：贝纳斯克，西班牙韦斯卡

# 舒适的西班牙村庄（© Miscelleneoustock/Alam）

当大雪覆盖着比利牛斯山脉陡峭的山坡时——这片山脉形成了西班牙和法国之间的天然边界——舒适的西班牙村庄Benasque就成了欧洲冬夜愉快的避难所。狭窄的鹅卵石街道两旁排列着保存完好的罗马式和文艺复兴时期的庄园和教堂，你很容易就会觉得自己仿佛回到了过去。除了这些文化魅力，大多数游客来Benasque是为了户外冒险。贝纳斯克山谷被比利牛斯山脉的最高峰所环绕，每年的这个时候都会下大雪，是一个受欢迎的滑雪胜地。当徒步旅行、山地自行车运动、滑翔伞和河流漂流成为吸引游客的主要因素时，夏天吸引了更多的游客来到这里。

# 2020 12月 10：正在睡觉的北极狐

# 狐狸做了什么梦?（© Menno Schaefer/Getty Image）

我们相信你可以像挤压毛绒玩具一样挤压这只打盹的北极狐，但我们不建议这样做。这些可爱的小毛球是凶猛的猎手。事实上，北极狐是苔原上最多产的捕食者之一，捕食旅鼠、田鼠、兔子、鸟类(通常有一侧卵)和其他经过它们可爱道路的小动物。但是现在，我们的北极狐正高兴地用它的尾巴蜷缩起来，睡上一觉——通过把身体蜷成一团，狐狸将尽可能少的表面积暴露在冷空气中，不管气温降到多低，都有助于保持热量。

# 适宜皇后的山脉（© Cyril Ruoso/Minden Picture）

# 2020 12月 18：从环绕山峰的小径最高点看到的拉瓦莱多三峰山，意大利

# “苍白的山脉”高耸的景色（© AWL Images/Danita Delimon）

看一下Tre Cime di Lavaredo，这是意大利阿尔卑斯山白云石山脉最具标志性的山峰之一。白云石，又名“苍白山”，因锯齿形山脉的浅色石头(白云石)而得名。这是从一条小径的最高点看到的，这条小径环绕着这三个引人注目的山峰。通常在夏季游人如云，秋季和春季则有机会沿着这9.6公里长的步道悠闲漫步，感受更宁静的氛围。徒步旅行的每一个弯道都展现出一个又一个令人惊叹的景色。沿途，徒步者将经过三个“rifugios”，这是阿尔卑斯山的传统庇护所，提供饮料和食物，甚至还有过夜的床铺。

# 2020 12月 19：清晨结了霜的树林，中国四姑娘山

# 四姐妹，千树万树（© Robert Harding World Imagery/Offset by Shutterstoc）

四姑娘山是中国西部邛崃山脉的最高峰，你会在四姑娘山的山坡上看到这幅被雪覆盖的森林景观。它的名字，翻译过来是“四姐妹山”，灵感来自于它的四个不同山峰背后的当地藏族传说。据传说，四姐妹用魔镜把自己变成了山里的魔鬼，救了自己的百姓。最高的山峰，“最小的妹妹”峰，高约6200米，是一个极具挑战性的攀登——很少有人尝试，直到1981年才有人尝试。周围的公园包括山的两侧的三个山谷，是四川大熊猫保护区的一部分，被联合国教科文组织列为世界遗产。除了大熊猫，该保护区还是小熊猫、雪豹、云豹以及5000至6000种植物的家园。

# 2020 12月 20：蒙大拿州西部的小雪羊

# 在雪地里穿一整天的衣服（© Donald M. Jones/Minden Picture）

山羊原产于北美，一年中大部分时间都生活在攀登从阿拉斯加中南部到落基山脉北部的山峰上。它们往往生活在海拔较高的地方，通常在3900米以上，它们稳健的攀爬能力使它们能够爬上异常陡峭的岩石斜坡。冬天，它们会迁移到海拔稍低的地方，在亚高山森林中寻找庇护，那里有朝南的岩石岩壁，那里的阳光和无情的风将冰降至最低。在这里，他们觅食地衣，苔藓，草和其他冬天的绿色植物。它们毛茸茸的双层皮毛让它们在北半球气温下降的时候感到温暖，这可能就是为什么即使在美国蒙大拿州的寒冷天气里，这只年轻的山山羊(或小山羊)似乎在玩球。

# 2020 12月 21：凯西克附近的Castlerigg石圈，英国坎布里亚

# 比巨石阵?（© Alan Copson/plainpictur）

大约5000年前，不列颠群岛的古代居民以某种方式拖着多达40块巨大的石头——最重的估计有16吨重——来到了现在英国坎布里亚郡湖区国家公园的这片草地高原。然后，他们把它们分组到卡斯勒里格的石头圈里，这里可以看到北半球冬季低太阳投下的影子。考古学家认为，石头圈的排列与太阳和月球的位置一致。它们被用在精心制作的仪式中来庆祝一些场合，比如今天的冬至，在地球的另一边，冬至是一年中白昼最短(黑夜最长)的日子。

# 2020 12月 23：莫阿尔博阿尔附近的带纹矛吻海龙，菲律宾

# 游泳进入季节（© Jenna Szerlag/Alam）

这种水里的甘蔗糖被称为条纹管鱼。你不会在北极或圣诞树上找到它，而是在印度-太平洋地区的热带海洋，比如澳大利亚、日本、菲律宾和南非。它和海马属于同一科，和它的表亲一样，管鱼的身体上覆盖着一层层骨质的盔甲。这给了它保护，但它是一个坚硬的身体(像棒棒糖一样!)，所以它通过快速扇动鱼鳍来游泳。也像海马一样，是雄性管嘴鱼而不是雌性管嘴鱼产卵。经过一段精心制作的求偶舞后，雌企鹅会把卵放在雄企鹅的育儿袋里，在那里发育直到雄企鹅分娩。我们并不是在编造这些东西，但我们不能保证红白相间的条纹管鱼有薄荷味的理论。

# 2020 12月 24：挪威苔原上的北极光和野生驯鹿

# 你们不是有地方可去吗?（© Anton Petrus/Getty Image）

驯鹿是圣诞季节最受认可的生物之一，它们生活在像挪威北部这样寒冷的北极地区，在令人惊叹的北极光漩涡下拍摄了这群驯鹿的照片。驯鹿进化出独特的特征，帮助它们在恶劣的环境中茁壮成长。它们的鼻子适应在空气进入肺部之前加热它们吸入的空气，这有助于即使在最冷的天气也能保持稳定的体温。它们的大部分身体，包括蹄子，都覆盖着厚厚的皮毛，以使它们很好地隔热。事实上，他们更有可能过热而不是结冰——所以当他们今晚在星空中飞行时，他们会感到比穿着红色天鹅绒衣服的圣诞老人更温暖。

# 在一棵树上栖息。不是你想的那样（© Michel Rauch/Minden Picture）

# 2020 12月 29：卢塞恩老城，瑞士

# 假期的光辉在哪里徘徊（© Xantana/Getty Image）

在卢塞恩湖畔，你会发现与湖同名的中世纪瑞士城市。卢塞恩始建于公元750年，是瑞士中部最大的城镇。卢塞恩古城的三座古老的木制步行桥的中心有一个独特的特点。Kapellbrucke(教堂桥)的内部绘有宗教场景和寓言，可以启发人们从桥上走到圣彼得教堂。虽然我们正在休假，但这个说德语的城市才刚刚开始。圣诞节和新年之后，开始为Fasnacht(瑞士和德国的狂欢节)做准备。就像狂欢节一样，街道上挤满了数百名戴着奇异面具的喧闹的游行者，他们在大斋节的斋戒前释放自己。
